# سجل التغييرات

## 11.0.0+23 - 2026-07-11

### أضيف

- مركز قرار محلي يرتب التجارب المنتهية، والتجديدات القريبة بلا استخدام، وارتفاعات الأسعار، والخدمات التي تحتاج مراجعة.
- لقطة تدفق نقدي في الرئيسية توضح خصومات 7 و30 يومًا والتجارب القريبة.
- فلاتر للأولوية وإجراءات مباشرة لفتح الاشتراك أو تسجيل استخدامه.
- حالة مزامنة واضحة مع وقت آخر نجاح داخل الإعدادات.
- اختبارات لوحدة ذكاء التجديد والقرارات.

### حُسّن

- حفظ حالة كل تبويب وموضع التمرير عند التنقل باستخدام `IndexedStack`.
- الوضع الافتراضي للمستخدم الجديد يتبع إعداد النظام، مع احترام اختيار المستخدمين الحاليين.
- تخطيطات v11 تستجيب للشاشات الضيقة وتحافظ على RTL كامل.
- فحص CI يحلل مصادر التطبيق والاختبارات قبل بناء iOS.

### التوافق والأمان

- لا تغيير على مخطط JSON الحالي أو معرّفات الاشتراكات؛ بيانات الإصدارات السابقة تُقرأ كما هي.
- لا تغيير على `firestore.rules` أو صيغة النسخة السحابية أو تشفير AES-256-GCM.
- لم تُضعف حماية المصادقة أو App Check أو حدود الاستيراد والنسخ السحابي.

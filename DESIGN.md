# تصميم اشتراكاتي v12

## الاستكشاف

### 1. مدار التجديد

كل اشتراك نقطة على مدار زمني؛ قرب النقطة من موضع «الآن» يعني قرب الخصم، وحجمها يعكس التكلفة السنوية.

```text
┌──────────────────────────┐
│  مساء الخير       بحث ◉ │
│                          │
│       ·      ●           │
│    ◌    ١٢ شهرًا   ◉     │
│       8,420 ر.س          │
│  ●          ·            │
│                          │
│ هذا الأسبوع   التسرّب    │
│ Netflix  55   1,140 ر.س │
└──── الرئيسية · مكتبتك ──┘
```

### 2. خريطة الكتل

مساحة كل خدمة تساوي تكلفتها السنوية، لتظهر الخدمات الكبيرة فورًا بلا قراءة أرقام كثيرة.

```text
┌──────────────────────────┐
│ خريطة التزامك السنوي     │
│ ┌───────────┬──────┐     │
│ │ STC       │Claude│     │
│ │           ├───┬──┤     │
│ │           │Nfx│  │     │
│ └───────────┴───┴──┘     │
│ اسحب كتلة للمراجعة       │
└──────────────────────────┘
```

### 3. واجهة الأوامر

لا وجهات ثابتة؛ يبدأ التطبيق ببحث وأوامر مثل «أرني تجديدات الأسبوع» و«أضف اشتراكًا».

```text
┌──────────────────────────┐
│ ماذا تريد أن تفعل؟       │
│ ┌──────────────────────┐ │
│ │ ابحث أو نفّذ أمرًا…  │ │
│ └──────────────────────┘ │
│ القادم  •  غير المستخدم │
│ + إضافة   ↑ استيراد      │
└──────────────────────────┘
```

## الاتجاه المختار

اختير **مدار التجديد** لأنه يجمع المال المتكرر والزمن في قراءة واحدة، ويمنح التطبيق عنصرًا لا يصلح لتطبيق مصرفي عام. استُخدمت واجهة الأوامر كطبقة تنقل مساعدة، بينما استُبعدت خريطة الكتل من الشاشة الأولى لأنها تضعف دقة المقارنة عند وجود عملات متعددة وتحتاج مساحة أكبر من شاشة الهاتف.

## نقد ذاتي

كانت النسخة الأولى من الخطة تحتوي صفًا من بطاقات KPI يشبه أي لوحة مالية. استُبدل بمدار واحد، وشريط «تسرّب صامت» خطي، وقائمة تجديدات بلا حاويات متداخلة. أُبقيت الأرقام الثانوية هادئة حتى لا تنافس العنصر التوقيعي.

## نظام الرموز

### الألوان

| الرمز | فاتح | داكن | الاستخدام |
|---|---|---|---|
| `ink` | `#13231D` | `#F2F7F4` | النص الأساسي |
| `canvas` | `#F4F7F3` | `#0D1411` | خلفية التطبيق |
| `surface` | `#FFFFFF` | `#151F1A` | سطح الأدوات |
| `pulse` | `#007D5C` | `#47D7A5` | الهوية والحالة النشطة |
| `amber` | `#D79A2B` | `#F0C56A` | موعد قريب/تنبيه |
| `coral` | `#C94F5C` | `#FF8D97` | تسرّب أو خطر مالي |

التباين بين `ink` و`canvas` يتجاوز 4.5:1، ولا يعتمد أي معنى على اللون وحده.

### الخطوط

- العرض: **Tajawal** بوزن 700/800 عند توفره، لأنه هندسي وواضح في الأرقام العربية.
- النص: **IBM Plex Sans Arabic** بوزن 400/600 عند توفره، لارتفاع سطر مريح وكثافة جيدة.
- لا يجلب التطبيق خطوطًا من الشبكة. تستخدم المنصات fallbacks عربية محلية (`SF Arabic`, `Geeza Pro`, `Noto Sans Arabic`) حتى تُضاف ملفات الخط المرخصة لاحقًا دون كسر الإقلاع.

### المقاييس

- المسافات: `4, 8, 12, 16, 24, 32, 48`.
- الخط: `12, 14, 16, 20, 28, 36` مع احترام Text Scale.
- أنصاف الأقطار: `4, 8, 14`؛ ويُحجز `14` للمدار ولوحة الأوامر.
- أهداف اللمس: `44pt` كحد أدنى.
- الطبقات: حد 1px وظل واحد منخفض؛ لا توجد بطاقات داخل بطاقات.

## العنصر التوقيعي

**مدار التجديد**: CustomPainter يرسم دورة 12 شهرًا، وموضع «الآن»، وأقرب التجديدات كنبضات متفاوتة الحجم حسب التكلفة السنوية. له ملخص Semantics نصي، ويتوقف انتقاله عند تفعيل تقليل الحركة.

## الحركة

المشهد الوحيد البارز هو دخول المدار والعناصر القريبة بتلاشي وانزلاق قصير 360ms. بقية الواجهة تستخدم انتقالات ضمنية 180ms فقط، وتصبح فورية عند `disableAnimations`.

## التخطيط والتنقل

- الهاتف: محتوى ممتد ثم `CycleDock` رمزي غير عائم، مع زر أوامر مركزي.
- iPad: شريط أوامر جانبي بعرض ثابت ومحتوى ثنائي الأعمدة.
- لوحة الأوامر تصل إلى جميع الوجهات والإضافة والاستيراد والإعدادات.
- لا يستخدم v12 `BottomNavigationBar` أو `NavigationBar`.

## الميزات المختارة

- لوحة التسرّب المالي والتكلفة السنوية الحقيقية.
- مدار التجديد والتنبيهات المحلية الحالية بمنطق المنطقة الزمنية الموجود.
- إبراز الاشتراكات بلا استخدام وتقسيم التكلفة العائلية.
- لوحة أوامر وبحث موحدة.
- تخطيط iPad متكيف وتحليلات CustomPainter الحالية.

## ما استُبعد

- النسخ الاحتياطي اليدوي: مستبعد احترامًا لقرار المنتج السابق وتقليلًا لمسارات تسريب البيانات؛ تبقى المزامنة الحالية بإفصاحها الأمني.
- Widget/شاشة القفل: يحتاج App Group وامتدادًا أصليًا ودورة مراجعة مستقلة، ولا يُمزج مع إعادة تصميم حرجة.
- مقارنة سنة بسنة: لا يوجد دفتر حركات تاريخي كامل؛ عرض رقم مصطنع سيكون مضللًا.

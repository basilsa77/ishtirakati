# اشتراكاتي 15.2.1+36

تطبيق Flutter عربي أولًا لإدارة الاشتراكات والأقساط والفواتير في السعودية والخليج. يعمل محليًا دون حساب، ويدعم مزامنة Firebase الاختيارية.

## ما الجديد في v15

- واجهة عربية وإنجليزية كاملة مع تبديل فوري بين RTL وLTR وخيار اتباع لغة iPhone.
- سلّم تايبوغرافي موحد بخط IBM Plex Sans Arabic للأحرف العربية واللاتينية، دون أحجام خطوط رقمية متناثرة.
- توطين التواريخ والأرقام والعملات والتنبيهات ورسائل Firebase والتحليل الذكي.
- مزامنة Firestore ذات إصدارات تمنع الكتابة فوق نسخة أحدث، مع اختبارات تعارض وانقطاع واسترجاع على المحاكي.
- تخطيط iPhone جديد يمنع تداخل الصفحات مع شريط التبويبات ويزيل حالات الشاشة الرمادية غير المفهومة.
- دعم متزن لحجم النص الكبير في iOS مع بقاء العناوين والبطاقات والقوائم قابلة للاستخدام.
- تسجيل دخول ينتظر نتيجة Firestore الفعلية ويعرض سبب فشل المزامنة بدل رسالة اتصال عامة.
- Firebase App Check يستخدم App Attest مع DeviceCheck fallback الرسمي لأجهزة Apple.
- شاشة دخول Cupertino قابلة للتمرير ومهيأة للوضع الداكن والشاشات الصغيرة.
- لا تغيير على بيانات v13 أو AES-256-GCM أو مفاتيح Keychain أو قواعد عزل Firestore.

## ما أضيف في v14

- واجهات اختيار أصلية لـ iOS للعملة والتصنيف ووسيلة الدفع ومزود البريد.
- إعادة تنظيم شاشة الاشتراكات مع فرز ظاهر وفصل نوع الدفعة عن التصنيف.
- إضافة سريعة متكيفة مع لوحة المفاتيح وتعرض دورة التجديد والعملة بوضوح.
- إعدادات أقصر وأوضح مع حالة مزامنة واحدة ورسائل iOS أصلية.
- لا تغيير على مخطط بيانات v13 أو تشفير AES-256-GCM أو مفاتيح Keychain.

## ما أضيف في v13

- **مساعد مالي محلي:** توقع فعلي للتجديدات خلال 12 شهرًا، ورصد الاشتراكات المكررة وارتفاع السعر والخدمات المتجددة بلا استخدام.
- **مراجعات مالية قابلة للتنفيذ:** تسجيل الاستخدام أو إنهاء المراجعة من ورقة إجراءات iOS واضحة.
- **إضافة سريعة:** اسم ومبلغ فقط لإضافة اشتراك شهري، مع إبقاء نموذج التفاصيل الكامل متاحًا.
- **مقارنة الخطط:** توحيد السعر الشهري والسنوي لإظهار الوفر الحقيقي قبل تغيير الخطة.
- **تنبيهات أذكى وأكثر خصوصية:** لا تنبه لخدمة غير متجددة تلقائيًا، وتخفي الاسم والمبلغ على شاشة القفل افتراضيًا.
- **تنقل iOS أصيل:** `CupertinoTabBar` بخمس وجهات على iPhone وSidebar متكيف على iPad.
- **ترحيل آمن:** سجلات v12 تُقرأ وتوسم بمخطط v13 دون تغيير المعرّفات أو القيم المالية.
- **تحقق إنتاجي:** اختبارات واجهة للهاتف وiPad، واختبار Firestore Emulator لعزل المستخدمين.

## إرث v11 المحفوظ

- **مركز القرار:** يرتب ما يحتاج انتباهك بدل عرض الأرقام فقط.
- **تدفق 7/30 يومًا:** مبالغ ودفعات التجديد القريبة في الصفحة الرئيسية.
- **مراجعة ذكية محلية:** تجارب تنتهي، خدمات بلا استخدام، وارتفاعات أسعار.
- **تنقل ثابت:** كل تبويب يحتفظ بحالته وموضع التمرير.
- **مزامنة مفهومة:** حالة المحاولة ووقت آخر نجاح داخل الإعدادات.
- **مظهر تلقائي:** المستخدم الجديد يتبع الوضع الفاتح أو الداكن في iOS.

## المزايا

- اشتراكات وأقساط وفواتير بدورات أسبوعية وشهرية وربع سنوية وسنوية.
- ثماني عملات: SAR وAED وKWD وQAR وBHD وOMR وUSD وEUR، والريال افتراضيًا.
- تقويم تجديدات، تنبيهات محلية، تجارب مجانية، سجل أسعار، ميزانية شهرية، واستخدام الخدمة.
- استيراد محلي من رسائل البنك وإيصالات Apple، وربط IMAP اختياري لا يحفظ كلمة المرور.
- تصنيف محلي وكتالوج شبكي، مع AI اختياري بموافقة صريحة ومفتاح المستخدم.
- Google وApple Auth ومزامنة Firestore اختيارية.
- واجهة عربية RTL وإنجليزية LTR ووضعان فاتح وداكن.

## الخصوصية والأمان

- بيانات الاشتراكات المحلية مشفرة بـAES-256-GCM.
- قفل اختياري ببصمة الوجه يفشل مغلقًا ولا يتجاوز المصادقة عند الخطأ.
- Keychain هو المصدر الوحيد لإنشاء وحفظ مفاتيح AES وAI في v13. تُقرأ مرايا v12 القديمة مرة واحدة لإنقاذ البيانات، ثم تُحذف فقط بعد نجاح ترحيل المفتاح إلى Keychain.
- Firestore مقيد بالمستخدم المالك، وحجم النسخة ومخططها محدودان في `firestore.rules`.
- Firebase App Check مهيأ لـApp Attest مع DeviceCheck fallback، ويُفرض تدريجيًا من لوحة Firebase بعد مراقبة الطلبات الصالحة.
- لا يوجد ادعاء بتشفير سحابي طرفي؛ راجع الحدود وخطوات لوحة Firebase في `SECURITY_AUDIT.md`.
- حذف الحساب من داخل التطبيق يحذف مستند Firestore وحساب Firebase ثم البيانات المحلية بعد إعادة المصادقة.
- سياسة الخصوصية العربية في [PRIVACY_POLICY.md](PRIVACY_POLICY.md).

## التشغيل والتحقق

```bash
flutter pub get
flutter analyze lib test --no-fatal-warnings --no-fatal-infos
flutter test
flutter build ios --release --no-codesign --dart-define=ENABLE_FIREBASE_APP_CHECK=false
```

مجلد iOS يُولّد في CI. إعدادات Firebase غير موجودة في Git، وتُستعاد من GitHub Actions Secrets أثناء البناء. راجع `FIREBASE_CI_SETUP.md` و`BUILD_IPA_GUIDE.md`.

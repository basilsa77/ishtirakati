# 💳 اشتراكاتي (Ishtirakati)

**تتبّع اشتراكاتك الرقمية وتحكّم بمصروفك — عربي أولًا، للسوق السعودي والخليجي.**

تطبيق iOS (وقابل للتشغيل على أندرويد/ويب لاحقًا) مبني بـ Flutter، يجيب على سؤال
يغفل عنه معظمنا: *كم أدفع فعليًا كل شهر على شاهد ونتفلكس وiCloud والنادي والباقات؟*

## لماذا هذه الفكرة؟

- تقرير Self Financial (مايو 2026): المستخدم يملك في المتوسط **2.6 اشتراك
  غير مستخدم** يدفع ثمنه بلا فائدة (~26.79$ شهريًا ≈ **320$+ سنويًا هدرًا**) —
  ما يُعرف بـ"نزيف الميزانية الصامت".
- **62%** من المستخدمين يشعرون بالإرهاق من كثرة الاشتراكات المتداخلة
  (تقرير Frisbii) — "إعياء الاشتراكات" ظاهرة عالمية متصاعدة.
- الاشتراكات المنسية **خطر أمني** أيضًا: كل خدمة مهملة تحتفظ ببيانات الدفع
  وتوسّع سطح الهجوم عند أي اختراق.
- المنافسون الأقوياء (Rocket Money، Bobby...) **بالإنجليزية** وبدون خدمات
  المنطقة — لا يوجد لاعب عربي مهيمن بواجهة RTL وقوائم جاهزة لخدماتنا المحلية،
  وبعضهم يطلب ربط حسابك البنكي (مخاوف خصوصية) بينما «اشتراكاتي» محلي 100%.

## المزايا (الإصدار 1.0)

- لوحة رئيسية: المصروف الشهري والسنوي الحقيقي + التجديدات القادمة خلال ٣٠ يومًا
  بشارات تنبيه ملوّنة (اليوم / غدًا / بعد X أيام).
- إضافة سريعة من **٢٢ خدمة شائعة** سعوديًا وخليجيًا (شاهد VIP، stc tv، أنغامي،
  Jawwy TV، Careem Plus...) أو إدخال يدوي كامل.
- دورات فوترة: أسبوعي / شهري / كل ٣ أشهر / سنوي — مع حساب تجديد ذكي يحافظ على
  يوم الشهر الأصلي (اشتراك يوم 31 يتجدد 28 فبراير ثم يعود 31 مارس).
- ٨ عملات خليجية وعالمية (الريال افتراضيًا) مع إجماليات مفصولة لكل عملة.
- إيقاف مؤقت لأي اشتراك مع إظهار **كم توفّر شهريًا** بإيقافه.
- تحليلات: توزيع الإنفاق بالتصنيفات (رسم أشرطة)، أغلى ٥ اشتراكات، توقع سنوي.
- بحث وفلترة بالتصنيف، حذف بالسحب مع تأكيد.
- **خصوصية كاملة**: كل البيانات محليًا على الجهاز (SharedPreferences) —
  لا خوادم، لا حسابات، لا تتبّع.

## نموذج الربح

| المرحلة | الوصف |
|---|---|
| مجاني | كل مزايا 1.0 أعلاه — لبناء قاعدة مستخدمين وتقييمات |
| Pro (اشتراك) | تنبيهات قبل التجديد، نسخ احتياطي iCloud، تقارير PDF، تحويل عملات — مقترح 7.99 ر.س/شهر |
| التوسع | نفس التطبيق يخدم الخليج فورًا (عملاتهم مدعومة)، ثم عربيًا |

## بنية المشروع

```
ishtirakati/
├── lib/
│   ├── main.dart                    # نقطة الدخول + التنقل السفلي
│   ├── theme.dart                   # الهوية (زمردي/رملي) + مكوّنات مشتركة
│   ├── models/subscription.dart     # النموذج + منطق التجديد والتكاليف
│   ├── data/presets.dart            # الخدمات الشائعة والتصنيفات
│   ├── services/subscription_store.dart  # الحفظ المحلي + الإحصائيات
│   └── screens/
│       ├── dashboard_screen.dart
│       ├── subscriptions_screen.dart
│       ├── edit_subscription_screen.dart
│       ├── insights_screen.dart
│       └── settings_screen.dart
├── test/subscription_test.dart      # اختبارات منطق التجديد والترميز
├── .github/workflows/build-ipa.yml  # بناء IPA سحابيًا على macOS
├── BUILD_IPA_GUIDE.md               # ← ابدأ من هنا للتثبيت على آيفونك
└── pubspec.yaml
```

ملاحظة: مجلدا `ios/` و`android/` غير مرفوعين عمدًا — يولّدهما
`flutter create` تلقائيًا داخل CI (وعلى جهازك عند الحاجة)، فيبقى المستودع نظيفًا.

## التشغيل والاختبار

راجع **BUILD_IPA_GUIDE.md** — دليل كامل بالعربية من ويندوز حتى التثبيت
على iPhone وiPad بتوقيع حسابك المطوّر.

للتطوير المحلي على ويندوز:

```bash
flutter create . --platforms=windows,web
flutter pub get
flutter test
flutter run -d chrome
```

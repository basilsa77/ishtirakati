# 📲 دليل بناء وتوقيع وتثبيت «اشتراكاتي» — من ويندوز إلى iPhone 17 Pro Max

هذا الدليل مكتوب خصيصًا لحالتك: **جهاز ويندوز + حساب مطوّر Apple مدفوع + آيفون وآيباد**.
لا تحتاج ماك إطلاقًا — البناء يتم سحابيًا على GitHub، والتوقيع والتثبيت من ويندوز.

> الفكرة باختصار: GitHub Actions يبني لك ملف IPA غير موقّع على أجهزة macOS السحابية المجانية،
> ثم برنامج **Sideloadly** على ويندوز يوقّعه بحسابك المطوّر ويثبّته على أجهزتك مباشرة.
> بما أن حسابك مطوّر مدفوع، يبقى التطبيق صالحًا **سنة كاملة** (وليس ٧ أيام كالحسابات المجانية).

---

## المتطلبات (مرة واحدة فقط)

1. حساب GitHub (مجاني) — عندك مسبقًا: basilsa77.
2. تثبيت Git على ويندوز: من git-scm.com (خيارات التثبيت الافتراضية تكفي).
3. تثبيت **Sideloadly** على ويندوز: من sideloadly.io (نسخة Windows 64-bit).
4. تثبيت **Apple Devices** من متجر Microsoft Store (أو iTunes من موقع Apple) —
   لتعريفات USB الخاصة بالآيفون.
5. كيبل USB للآيفون/الآيباد.

## الخطوة 1: رفع المشروع إلى GitHub

افتح PowerShell داخل مجلد المشروع `ishtirakati` ونفّذ:

```powershell
git init
git add .
git commit -m "الإصدار الأول من اشتراكاتي"
git branch -M main
git remote add origin https://github.com/basilsa77/ishtirakati.git
git push -u origin main
```

> أنشئ المستودع `ishtirakati` أولًا من github.com → New repository (بدون README).
> إن طلب منك تسجيل الدخول، استخدم Personal Access Token بدل كلمة المرور
> (GitHub → Settings → Developer settings → Tokens).

---

## الخطوة 2: تشغيل البناء السحابي

1. افتح المستودع على GitHub → تبويب **Actions**.
2. إن ظهر زر تفعيل Workflows فعّله.
3. اختر **Build iOS IPA** من القائمة اليسرى → زر **Run workflow** → **Run workflow**.
4. انتظر ٨–١٥ دقيقة حتى تظهر ✅ خضراء.
   (البناء يشغّل الاختبارات تلقائيًا؛ لو فشل شيء سترى السبب في السجل.)
5. ادخل على التشغيل الناجح → قسم **Artifacts** بالأسفل → حمّل
   **Ishtirakati-15.0.2-build31-unsigned-ipa** → فك الضغط لتحصل على
   `Ishtirakati-15.0.2-build31-unsigned.ipa`.

> ملاحظة: أي `git push` جديد على main سيعيد البناء تلقائيًا.

---

## الخطوة 3: التوقيع والتثبيت من ويندوز عبر Sideloadly

1. وصّل **الآيفون** بالكمبيوتر عبر USB، وافتح القفل، واضغط **"الوثوق بهذا الكمبيوتر"** إذا ظهرت.
2. افتح Sideloadly — يجب أن يظهر اسم جهازك في القائمة العلوية.
3. اسحب ملف `Ishtirakati-15.0.2-build31-unsigned.ipa` إلى نافذة البرنامج.
4. في خانة **Apple account** أدخل بريد **حسابك المطوّر** المدفوع.
5. اضغط **Start** → أدخل كلمة المرور → أدخل رمز التحقق الثنائي عند طلبه.
6. انتظر حتى تكتمل الشريط ويظهر Done — ستجد أيقونة **اشتراكاتي** على الشاشة الرئيسية.

### أول تشغيل فقط: الوثوق بالمطوّر
الإعدادات → عام → **إدارة VPN والأجهزة** (VPN & Device Management)
→ اضغط على حسابك تحت Developer App → **الوثوق**.

### الآيباد
كرّر الخطوة 3 نفسها مع توصيل الآيباد — نفس الـ IPA يعمل على الجهازين
(المشروع مضبوط Universal: آيفون + آيباد).

## استكشاف الأخطاء الشائعة

| المشكلة | الحل |
|---|---|
| Sideloadly لا يرى الجهاز | تأكد من تثبيت Apple Devices/iTunes، غيّر الكيبل/المنفذ، وافتح قفل الجهاز |
| خطأ في تسجيل الدخول 2FA | استخدم كلمة مرور خاصة بالتطبيقات من appleid.apple.com → Sign-In and Security → App-Specific Passwords |
| "Unable to install" | احذف أي نسخة قديمة من التطبيق ثم أعد المحاولة |
| التطبيق لا يفتح بعد التثبيت | لم تنفّذ خطوة "الوثوق بالمطوّر" أعلاه |
| فشل البناء في Actions | افتح سجل الخطوة الحمراء وأرسل لي نص الخطأ كما هو |

## عند الجاهزية للنشر الفعلي (متجر التطبيقات)

مسار Sideloadly ممتاز للتجربة الشخصية، لكن للتوزيع والربح ستحتاج:

1. **Codemagic** (codemagic.io — خطة مجانية ٥٠٠ دقيقة macOS شهريًا):
   اربط المستودع → أضف مفاتيح App Store Connect API → يبني ويوقّع ويرفع
   مباشرة إلى **TestFlight** بدون ماك.
2. من TestFlight تجرّب مع مختبرين حقيقيين، ثم ترسل للمراجعة والنشر على App Store.
3. قبل النشر: تحقق من توفر اسم «اشتراكاتي» في App Store Connect،
   وجهّز أيقونة 1024×1024 ولقطات شاشة (أستطيع تصميمها لك لاحقًا).
4. الربح: فعّل In-App Purchase لاشتراك النسخة الاحترافية
   (مقترح: 7.99 ر.س/شهر أو 59 ر.س/سنة) — Apple تتكفل بالدفع والفوترة.
